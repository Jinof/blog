我们总是看到 AI 公司发布模型时会提到模型自主工作了 XX 小时，完成了 XX 工作, 那么他们是怎么让 AI 长时间自主工作的呢?
> https://z.ai/blog/glm-5.1: glm-5.1 发布博客中的视频中展示了连续 8 小时, 完成 Linux 桌面的能力. 视频中的 Coding Agent 似乎不是开源产品.
> https://mimo.xiaomi.com/mimo-v2-5-pro: mimo-v2.5-pro 发布博客中提到, 其花费 4.3 小时, 实现了 SysY Compiler. 没有关于 Coding Agent 的信息.
> https://www.kimi.com/blog/kimi-k2-6: kimi-k2.6 发布博客中提到, 其花费 12 小时, 成功在本地 Mac (M3 Max) 上部署了 Qwen3.5-0.8B, 并且优化推理速度从 ~15 到 ~193 token/s. kimi-k2.6 发布时的构建工具推测时是 kimi code / kimi swarm.
