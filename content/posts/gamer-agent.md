# Gamer Agent
游戏的爽感来自于用户做决策/操作后的正反馈，例如完成一个任务、触发一个好解决，预测出对手的下一步操作。而 LLM 具有很强的推导能力，可以实现一个 Agent 协助用户决策。

# 功能设计
针对游戏类型提供不同的功能:
对局游戏: 按照时间, 关键时间点提供策略
ARPG: 快捷键触发, 截图分析, 然后给玩家分析各种选项的后果.
...

先做 ARPG, 比较简单，并且容易有节目效果

# 架构设计
## MVP 版本
桌面 App -> 监听快捷键 -> 触发截图事件 -> 消费到截图事件 -> Agent 进行处理 -> 输出结果: 文本/语音/修改截图
MVP 版本不使用任何框架, 采用原生的开发方式. 首先支持 Macos 版本。
